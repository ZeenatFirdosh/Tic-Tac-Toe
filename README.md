## How to Use

Clone or download repo
NodeJS / NPM / Yarn should be installed in your PC

Open terminal or Git for Windows (Git Bash)
Run these commands:

### Install Dependencies

yarn install

OR

npm install

### Run app

yarn start

OR

npm start
